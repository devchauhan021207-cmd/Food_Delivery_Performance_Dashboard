# Food_Delivery_Performance_PowerBI
Interactive Food Delivery Performance Dashboard built using Power BI to analyze delivery time, traffic, weather, delivery modes, and peak order trends.
🍔 Food Delivery Performance Dashboard
📌 Project Overview

This project analyzes food delivery operations using Power BI. The dashboard helps identify delivery performance trends based on traffic, weather, delivery mode, and peak ordering times.

🎯 Objectives
Analyze delivery performance
Identify peak order periods
Compare delivery modes
Measure weather and traffic impact
Build an interactive dashboard
📊 KPIs
Total Orders
Average Delivery Time
Average Distance
Peak Order Time
📈 Dashboard Insights
Cars and bikes recorded lower delivery times than bicycles.
Cloudy weather had the highest average delivery time.
Morning was the busiest ordering period.
Traffic had only a small impact on average delivery time.
